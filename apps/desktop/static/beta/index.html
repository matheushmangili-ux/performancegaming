ClockReaper Setup (recomendado) NSIS · 2.4 MB · Windows 11 x64 ClockReaper MSI Windows Installer · 3.3 MB · pt-BR
O Windows vai mostrar um aviso "Windows protegeu seu PC". Clique em Mais informações → Executar assim mesmo. Não tem vírus, só falta assinatura digital (chega na semana 11).

Como testar

  1. Baixe o instalador acima.
  2. Se o Defender bloquear: Mais informações → Executar assim mesmo.
  3. Conclua a instalação (Next → Install → Finish).
  4. Abra pelo Menu Iniciar. Importante: para o scan funcionar, clique com o botão direito em ClockReaper → Executar como administrador.
  5. Clique Run System Scan e aguarde os findings chegarem.
  6. Explore: expanda findings, troque Profile (Safe/Balanced/Competitive) e Mode (Gaming/General/All).
